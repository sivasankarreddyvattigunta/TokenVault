Minting NFTs on Ethereum Using IPFS and Pinata

Overview

This document provides a detailed guide on how to mint NFTs on the Ethereum blockchain using IPFS for storage, specifically leveraging Pinata Cloud for easy management of your files. It will also discuss the process of creating a CID (Content Identifier) using Pinata and outline the security mechanisms that ensure the integrity of your NFT minting process.

1. Minting Process Overview

The process of minting an NFT involves several key steps:

  • Prepare the digital asset you want to mint.
  • Upload the asset to IPFS via Pinata.
  • Create a CID for the uploaded asset.
  • Mint the NFT on the TokenVault using the CID.

3. Creating a CID in Pinata

After your file is uploaded to Pinata, it will generate a CID (Content Identifier) for your asset. This CID is a unique identifier that allows you to reference the content stored on IPFS.

Steps to Obtain a CID:

  1. Once the file is uploaded, navigate to your files in the Pinata dashboard.
  2. Find the uploaded file, and you will see the generated CID next to it.
  3. Copy the CID, as you will need it for minting the NFT.
